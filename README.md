# ScholarArena

**Tournament Management Platform for Academic Competition Organizations**

ScholarArena is a comprehensive software solution designed specifically for organizations running academic competitions such as History Bee, Science Bee, Geography Bee, and Bowl tournaments.

## 🎯 Built For Organizations Like IACompetitions

This platform addresses the unique challenges of running large-scale academic competitions:

- **Multi-tournament management**: Run 150+ regional tournaments simultaneously with different question sets (Red, White, Blue, Gold) and divisions (Elementary/Middle School, Varsity/Junior Varsity)
- **Three-stage competition workflow**: Seamlessly manage qualifying exams → regional tournaments → national championships
- **Staff coordination**: Efficiently assign readers, scorekeepers, and operational staff across multiple venues
- **Real-time engagement**: Live brackets and scoring that families and participants can follow from anywhere

## ✨ Key Features

### For Tournament Directors
- **Registration & Payment**: Automated online registration with payment processing, waitlists, and confirmation emails
- **Multi-division Support**: Manage K-8, 9-12, and other custom divisions in parallel
- **Question Set Management**: Organize multiple question sets (A, B, C levels; Red, White, Blue, Gold tiers)
- **Staff Management**: Coordinate tournament readers, scorekeepers, and operational teams
- **Qualification Tracking**: Automatic tracking of who qualifies for nationals based on tournament performance

### For Participants & Families
- **Live Updates**: Real-time bracket updates and scoring accessible on any device
- **Family Portal**: Dedicated access for parents and coaches to view schedules, results, and standings
- **National Rankings**: Comprehensive leaderboards and historical performance tracking
- **Notifications**: Automated alerts for schedule changes, results, and important announcements

### Technical Excellence
- **99.9% Uptime**: Enterprise-grade reliability for critical tournament days
- **Buzzer System Integration**: Support for live buzzer-based scoring during tournaments
- **COPPA Compliant**: Full compliance with student data protection regulations
- **White-Label**: Fully customizable branding to maintain your organization's identity
- **Scalability**: Proven to handle 2,000+ participants at national championships

## 🚀 Tech Stack

- **Frontend**: Next.js 16, React, TypeScript
- **UI Components**: Radix UI, Tailwind CSS
- **Performance**: Server-side rendering, optimized for fast page loads
- **Analytics**: Built-in analytics with Vercel Analytics
